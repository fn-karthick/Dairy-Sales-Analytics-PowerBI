# Dairy Sales Analytics Dashboard
## Project Overview

This project presents an interactive Business Intelligence dashboard developed using Microsoft Power BI to analyze dairy product sales, revenue trends, customer purchasing patterns, inventory metrics, and regional performance.

The dashboard enables data-driven decision-making through dynamic visualizations and KPI monitoring.

## Objectives

* Analyze total revenue and quantity sold
* Monitor dairy product performance
* Compare sales channels
* Identify top-performing locations
* Track inventory and reorder requirements
* Discover key revenue drivers

## Tools & Technologies

* Microsoft Power BI Desktop
* Power Query
* DAX
* CSV Dataset
* GitHub

### Product & Customer Insights

This dashboard page provides a comprehensive analysis of product performance and customer demand patterns. It highlights top-selling products, brand-wise revenue contribution, sales channel effectiveness, customer location trends, inventory status, and reorder requirements. Interactive visualizations enable stakeholders to identify high-performing products, optimize inventory management, and make data-driven business decisions to improve sales and operational efficiency.
